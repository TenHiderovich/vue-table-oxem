# oxem-studio

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Build docker
```
docker build -t oxem-studio .
```

### Start from docker
```
docker run -it -p 8080:8080 --rm --name oxem-studio oxem-studio
```
